# Saluto Personalizzato

## Scopo dello script:
E' possibile intercambiare i nomi di persone, cose o animali per avere un saluto personalizzato.

## Prerequisiti:
PHP


## Comandi di esecuzione con esempi:

1. php greet.php  
2. php greet.php Giovanni
3. php greet.php Zio


## Output atteso:

1. Ciao, Mondo!
2. Ciao, Giovanni!  
3. Ciao, Zio!
